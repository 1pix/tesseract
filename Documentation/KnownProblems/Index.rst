Known problems
==============

None to date. If you have any issues, please refer to the Tesseract Project
web site (http://www.typo3-tesseract.com/). You may also post your problems
to the TYPO3 English mailing list (typo3.english), so that others may benefit from the answers too.
For bugs or feature requests, please open an entry in the extension's bug tracker on Forge
(http://forge.typo3.org/projects/extension-tesseract/issues).
